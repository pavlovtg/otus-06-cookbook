# Workflow: add-report — добавление записи в REPORT.md

## Роль

- AI-агент (Cline), ведущий историю создания проекта в `REPORT.md`.

## Контекст

- `REPORT.md` — обязательный артефакт проекта согласно PRD-0001-MVP.md.
- Заполняется по мере работы, не в конце.
- Критерий сдачи: содержит описание AI-инструментов и минимум один показательный пример промта с разбором.
- Разделы `REPORT.md`:
  - `## AI-инструменты` — используемые AI-инструменты и их роль в проекте.
  - `## Проблемы и решения` — ключевые проблемы и способы их решения.
  - `## Удачные шаги` — что сработало хорошо.
  - `## Неудачные шаги` — что не сработало или потребовало переработки.
  - `## Примеры промтов` — показательные примеры промтов с кратким разбором.

## Задача

Создать Cline workflow `add-report`, который дополняет отчет в следующем порядке:

1. Прочитать `REPORT.md` (если файл существует).
2. Если файл не существует — создать с заголовком `# REPORT` и всеми разделами (пустыми).
3. Определить целевой раздел по описанию пользователя:
   - Упоминание инструмента / модели / AI → `## AI-инструменты`
   - Упоминание проблемы / ошибки / препятствия → `## Проблемы и решения`
   - Упоминание успеха / хорошего результата → `## Удачные шаги`
   - Упоминание неудачи / переработки / отката → `## Неудачные шаги`
   - Упоминание промта / примера / запроса → `## Примеры промтов`
   - Если раздел неоднозначен — спросить пользователя.
4. Добавить запись в конец целевого раздела в формате:

   ```markdown
   ### <Краткий заголовок> (YYYY-MM-DD)

   <Описание события, проблемы, решения или примера>.
   ```

5. Сохранить файл.

## Требования

- Язык записей: русский.
- Формат: Markdown, соблюдать [стандарт оформления](../../standards/markdown-code-style.md).
- Не удалять и не переписывать существующие записи.
- Дата — текущая дата в формате `YYYY-MM-DD`.
- Для раздела `## Примеры промтов` — сразу заполнить в виде таблицы (Ссылка на файл, Краткое описание, Разбор промта). Таблицу заполнить на основании промтов 01-markdown-code-style.md, 03-adr-monorepo.md, 04-branch-plan-standard.md, 05-contributing-guide.md.
