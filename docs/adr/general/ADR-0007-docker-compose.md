# ADR-0007: Docker Compose как среда развёртывания

- **Статус**: принят
- **Дата**: 2026-06-06

## Контекст

PRD требует, чтобы всё приложение (backend + frontend + БД) запускалось одной командой `docker compose up` в чистом окружении без дополнительной настройки. Это блокирующий критерий сдачи. Альтернативы: Kubernetes Minikube (избыточен для учебного проекта), ручной запуск сервисов (не соответствует требованию одной команды) — отклонены.

## Решение

Все сервисы (backend, frontend, PostgreSQL) описаны в `docker-compose.yml`. Приложение запускается командой `docker compose up` без предварительной настройки.

## Последствия

- Все переменные окружения задаются в `docker-compose.yml` или `.env`-файле, включённом в репозиторий.
- Миграции БД и загрузка seed-данных выполняются автоматически при старте.
