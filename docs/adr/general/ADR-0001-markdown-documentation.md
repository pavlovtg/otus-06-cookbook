# ADR-0001: Markdown как единый формат документации

- **Статус**: принят
- **Домен**: general
- **Дата**: 2026-06-06

## Контекст

Документация репозитория используется совместно людьми и AI-агентами. Нужен формат, который:

- читается без специальных инструментов,
- поддерживается системами контроля версий,
- понятен AI-агентам без дополнительной обработки.

## Рассмотренные варианты

- **Markdown** — текстовый, версионируемый, широко поддерживается.
- **AsciiDoc** — мощнее, но менее распространён среди AI-инструментов.
- **Wiki (Confluence, Notion)** — внешние системы, не версионируются в репозитории.

## Решение

Использовать Markdown как единый формат для всей документации репозитория.

## Последствия

- Документация хранится в репозитории рядом с кодом.
- AI-агенты читают документацию напрямую через файловую систему.
- Все документы подчиняются единому стандарту оформления.
