# ADR-0011: .NET 10 / C# как платформа backend

- **Статус**: принят
- **Дата**: 2026-06-07

## Контекст

Проекту требуется единая платформа для реализации всех backend-сервисов: бизнес-логика, аутентификация, API Gateway. Команда обладает компетенциями в .NET и C#. Альтернативы: Node.js/TypeScript (дублирует стек frontend, не покрывает компетенции команды по доменному моделированию), Go (слабее по DDD-инструментарию), Java/Spring Boot (команда без профильных компетенций) — отклонены.

## Решение

Все backend-сервисы реализуются на **.NET 10** на языке **C#** (последняя версия языка, поддерживаемая платформой).

## Последствия

- Единый стек упрощает CI, базовые шаблоны проектов, обмен компетенциями.
- Все разработчики backend должны владеть .NET 10 / C#.
- Иные платформы (Node.js, Go, Java и т. п.) для backend-сервисов запрещены без отдельного нового ADR.
- Требуется зафиксировать стандарт оформления C#-кода и структуры .NET-решений.
