# Стандарт оформления Markdown

## Базовые правила

- Кодировка: UTF-8
- Отступы в списках: 2 пробела
- Маркер ненумерованного списка: `-`
- Заголовки: ATX-стиль (`#`, `##`, `###`)

## Правила

### MD009 — Нет пробелов в конце строки

Строки не должны заканчиваться пробелами.

### MD012 — Нет нескольких пустых строк подряд

Между блоками допускается не более одной пустой строки.

### MD022 — Заголовки окружены пустыми строками

До и после каждого заголовка — одна пустая строка (кроме начала файла).

### MD032 — Списки окружены пустыми строками

До и после блока списка — одна пустая строка.

### MD034 — Нет «голых» URL

URL оформляются как ссылки (`[текст](url)`) или в угловых скобках (`<url>`).

### MD047 — Файл заканчивается переводом строки

Последний символ файла — `\n`.
