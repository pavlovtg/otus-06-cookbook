# AR-0033: Параметризация nginx-конфига через nginx templates

Источник: ADR-0020

## Правило

nginx-конфиг параметризуется через механизм nginx templates: файл с расширением `.template` монтируется в `/etc/nginx/templates/` — nginx-образ автоматически прогоняет `envsubst` при старте и кладёт результат в `/etc/nginx/conf.d/`.

## Запрещено

- Хардкодить хосты и порты upstream-сервисов непосредственно в `nginx.conf`.
- Монтировать статический `nginx.conf` напрямую, если конфиг содержит конфигурируемые параметры.
