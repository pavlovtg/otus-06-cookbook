# AR-0002: Swagger UI доступен по живому URL

- **Домен**: general
- **Статус**: активно

## Правило

Swagger UI должен быть доступен по живому URL в запущенном приложении; статический файл спецификации не является достаточным.

## Обоснование

PRD-0001 требует, чтобы преподаватель мог открыть Swagger UI в браузере и проверить эндпоинты интерактивно.
Статический файл (`openapi.json`) не позволяет выполнять запросы к API из интерфейса.

## Нарушение

- Swagger UI не запущен или недоступен после `docker compose up`.
- Документация API представлена только статическим файлом в репозитории.
- URL Swagger UI требует дополнительной настройки или ручного запуска отдельного сервиса.
