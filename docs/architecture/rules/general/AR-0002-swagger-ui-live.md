# AR-0002: Swagger UI доступен по живому URL

Источник: ADR-0006, ADR-0009

## Правило

Swagger UI доступен по живому URL в запущенном приложении. Статический файл спецификации не является достаточным.

## Запрещено

- Swagger UI недоступен после `docker compose up`.
- Документация API представлена только статическим файлом в репозитории.
- URL Swagger UI требует дополнительной настройки или ручного запуска отдельного сервиса.
