# AR-0001: Docker Compose Self-Contained

- **Домен**: general
- **Статус**: активно

## Правило

Приложение запускается командой `docker compose up` в чистом окружении без дополнительной настройки переменных окружения, ручных миграций или внешних зависимостей.

## Обоснование

Блокирующий критерий сдачи проекта (PRD-0001): преподаватель выполняет `git clone && docker compose up` и ожидает работающее приложение с seed-данными.
Любое требование ручных шагов делает проверку невозможной и отправляет проект на доработку.

## Нарушение

- Требование задать переменные окружения вручную перед запуском.
- Требование выполнить миграции или seed вручную (`docker exec ...`, `npm run migrate`).
- Зависимость от внешних сервисов, недоступных в чистом окружении (внешняя БД, сторонний API без mock).
- Приложение не запускается или падает при первом `docker compose up` без предварительной подготовки.
