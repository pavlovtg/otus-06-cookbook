# AR-0003: `.editorconfig` — обязательный источник кодстайла

## Правило

Кодстайл всех файлов в репозитории фиксируется в едином `.editorconfig` в корне репозитория.

- `.editorconfig` обязателен и един для всего монорепо.
- Любая команда/стек, добавляющая код в репозиторий, фиксирует свои правила оформления в этом файле — через секции по glob-паттернам (`[*.cs]`, `[*.ts]`, `[*.md]`, `[*.yml]` и т. п.).

## Запрещено

- Кодстайл стека описан только в README/wiki, а не в `.editorconfig`.
- `.editorconfig` в подпапке, переопределяющий корневые правила без явного обоснования.
- Отсутствие проверки соответствия `.editorconfig` в CI.
