# AR-0042: Изоляция Microservice-тестов

Источник: ADR-0033, ADR-0027

## Правило

- Каждый Microservice-тест поднимает собственный экземпляр БД через Testcontainers.
- HTTP-зависимости других сервисов мокируются через WireMock.Net.
- Тест не зависит от внешней сети и запущенного `docker compose`.

## Запрещено

- Использовать общую БД между тестами одного прогона.
- Обращаться к реальным upstream-сервисам из Microservice-тестов.
