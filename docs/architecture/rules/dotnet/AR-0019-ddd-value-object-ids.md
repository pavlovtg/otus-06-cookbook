# AR-0019: Идентификаторы агрегатов — типизированные Value Objects

Источник: ADR-0012

## Правило

Идентификаторы агрегатов и сущностей реализуются как `readonly record struct`:

- Содержат единственное поле `Guid Value`.
- Предоставляют фабричный метод `From(Guid value)` с проверкой на `Guid.Empty`.
- Предоставляют метод `New()` для генерации нового идентификатора.
- `ToString()` возвращает инвариантный формат `"D"` (`xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`).

## Запрещено

- Использовать `Guid`, `int`, `string` напрямую как идентификаторы агрегатов.
- Создавать идентификатор без валидации (обход `From()`).
