# AR-0005: Backend — только .NET 10 / C#

Источник: ADR-0011

## Правило

Все backend-сервисы и backend-утилиты в репозитории реализуются на **.NET 10** на языке **C#**.

## Запрещено

- Создание backend-сервиса на другом языке или рантайме (Node.js, Go, Java, Python и т. п.).
- Использование устаревшей или экспериментальной версии .NET без обновления ADR.
