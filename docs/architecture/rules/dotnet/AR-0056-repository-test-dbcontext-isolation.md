# AR-0056: Изоляция DbContext в тестах репозитория

Источник: ADR-0033

## Правило

В каждом тесте репозитория используется новый инстанс `DbContext`. Если тест выполняет запись, а затем читает результат через репозиторий — запись и чтение выполняются через разные инстансы `DbContext`. Подготовка БД в тесте также использует отдельный инстанс `DbContext`.

## Запрещено

- Переиспользовать один инстанс `DbContext` между операцией записи и последующей проверкой чтения.
- Использовать кешированные данные контекста для верификации реальной записи в БД.
