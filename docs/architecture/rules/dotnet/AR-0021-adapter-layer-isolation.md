# AR-0021: Изоляция адаптеров — адаптеры не зависят друг от друга

Источник: ADR-0013

## Правило

Каждый адаптер зависит только от слоёв Application и Domain:

- `Adapters/Web` импортирует только `Application` (входящие порты/сервисы) и `Domain` (для DTO-маппинга).
- `Adapters/Postgresql` импортирует только `Application` (исходящие порты) и `Domain`.

## Запрещено

- Импортировать типы из `Adapters/Postgresql` в `Adapters/Web` и наоборот.
- Любые прямые зависимости между адаптерами одного сервиса.
