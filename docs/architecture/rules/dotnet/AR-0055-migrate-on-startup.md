# AR-0055: Миграция схемы при старте приложения через MigrateDatabaseAsync

Источник: ADR-0014

## Правило

Миграция схемы выполняется на этапе конфигурирования приложения, до поднятия эндпоинтов, с использованием хелпера `MigrateDatabaseAsync<TProgram, TContext>` из `Shared.Hosting`.

## Запрещено

- Запускать миграцию после старта эндпоинтов.
- Вызывать `MigrateAsync` напрямую в коде приложения, минуя хелпер.
- Условно пропускать миграцию в зависимости от окружения (см. AR-0023).
