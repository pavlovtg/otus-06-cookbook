# AR-0049: Асинхронные методы репозитория с CancellationToken

Источник: ADR-0014

## Правило

Все методы интерфейсов репозиториев — асинхронные. Каждый метод принимает `CancellationToken` последним параметром и передаёт его во все операции ввода-вывода.

## Запрещено

- Объявлять синхронные методы в интерфейсе репозитория.
- Игнорировать переданный `CancellationToken` внутри реализации.
