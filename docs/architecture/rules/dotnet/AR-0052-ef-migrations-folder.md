# AR-0052: Миграции EF Core располагаются в папке Migrations

Источник: ADR-0014

## Правило

Все миграции EF Core располагаются в папке `Migrations/` внутри папки адаптера (или подпапки bounded context согласно AR-0047).

## Запрещено

- Размещать миграции вне папки `Migrations/`.
- Хранить миграции в отдельном проекте.
