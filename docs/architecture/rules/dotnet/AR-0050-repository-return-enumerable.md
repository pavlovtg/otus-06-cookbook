# AR-0050: Возвращаемые коллекции репозитория — IEnumerable или IAsyncEnumerable

## Правило

Методы репозиториев, возвращающие коллекции, используют `IEnumerable<T>` или `IAsyncEnumerable<T>`. Материализация в `List<T>` или массив допустима только если она необходима для реализации (например, EF Core требует `ToListAsync` перед закрытием соединения).

## Запрещено

- Объявлять возвращаемый тип `List<T>`, `T[]` или другой конкретный тип коллекции в интерфейсе репозитория.
