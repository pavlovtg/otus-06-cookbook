# AR-0057: BFF — один файл на ресурс

Источник: ADR-0017

## Правило

Каждый ресурс BFF (набор функций для одного REST-ресурса) располагается в отдельном файле `apps/web/lib/bff/<resource>.ts`.

## Запрещено

- Хранить функции для нескольких ресурсов в одном файле (например, `gateway.ts` с рецептами и ингредиентами).
- Создавать общий файл-агрегатор BFF-функций.
