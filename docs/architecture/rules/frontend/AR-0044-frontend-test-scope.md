# AR-0044: Область тестирования frontend

Источник: ADR-0033

## Правило

- Unit: Zod-схемы, утилиты `lib/`, BFF-агрегаторы с мокированным `fetch`.
- Component: ключевые UI-компоненты — рендер, события, состояния loading/error/empty.

## Запрещено

- Тестировать сторонний код (Next.js, React, библиотеки компонентов).
- Включать визуальную регрессию стилей в unit/component-тесты.
