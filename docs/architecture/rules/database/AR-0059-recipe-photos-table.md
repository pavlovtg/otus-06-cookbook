# AR-0059: Таблица recipe_photos — структура и владение

Источник: ADR-0034

## Правило

Фотографии рецептов хранятся в отдельной таблице `recipe_photos`, принадлежащей cookbook-сервису. Таблица содержит бинарные данные оригинала и thumbnail в колонках типа `bytea`. При удалении рецепта все связанные фотографии удаляются каскадно (`ON DELETE CASCADE`).

## Запрещено

- Хранить бинарные данные фотографий в таблице `recipes`.
- Обращаться к таблице `recipe_photos` из любого сервиса, кроме cookbook-сервиса.
