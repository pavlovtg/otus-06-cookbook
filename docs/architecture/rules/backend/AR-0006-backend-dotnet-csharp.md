# AR-0006: Backend — только .NET 10 / C#

- **Домен**: backend
- **Статус**: активно

## Правило

Все backend-сервисы и backend-утилиты в репозитории реализуются на **.NET 10** на языке **C#**.

Использование других платформ и языков (Node.js, Go, Java, Python и т. п.) для backend-сервисов запрещено без отдельного нового ADR, отменяющего или дополняющего [ADR-0011](../../../adr/backend/ADR-0011-dotnet-csharp-backend.md).

## Обоснование

Единый стек упрощает CI, шаблоны проектов, обмен компетенциями и сопровождение. Команда обладает компетенциями в .NET / C#.

## Нарушение

- Создан backend-сервис на другом языке/рантайме.
- Используется устаревшая или экспериментальная версия .NET без обновления ADR.
