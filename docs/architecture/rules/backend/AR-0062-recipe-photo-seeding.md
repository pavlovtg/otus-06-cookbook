# AR-0062: Seed-фотографии загружаются через CookbookSeeder

Источник: ADR-0034

## Правило

Seed-фотографии рецептов хранятся в репозитории в папке `apps/seed/photos/`. При первом запуске `CookbookSeeder` читает файлы из этой папки и загружает их в таблицу `recipe_photos` в рамках той же транзакции, что и seed-рецепты. Операция идемпотентна: повторный запуск не создаёт дублей.

## Запрещено

- Хранить seed-фотографии вне папки `apps/seed/photos/`.
- Загружать seed-фотографии отдельным скриптом или init-контейнером.
