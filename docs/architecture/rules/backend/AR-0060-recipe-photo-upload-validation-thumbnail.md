# AR-0060: Загрузка фото — серверная валидация и генерация thumbnail

Источник: ADR-0034

## Правило

При загрузке фото рецепта сервер обязан:

- Проверить MIME-тип файла: допустимы только `image/jpeg` и `image/png`.
- Проверить размер файла: не более 10 МБ.
- Сгенерировать thumbnail размером 400×300 px с обрезкой по центру (crop center) с помощью ImageSharp.
- Сохранить оригинал и thumbnail в одной строке таблицы `recipe_photos`.

## Запрещено

- Принимать файлы с MIME-типом, отличным от `image/jpeg` и `image/png`.
- Сохранять фото без предварительной серверной валидации.
- Использовать SkiaSharp или другие библиотеки обработки изображений вместо ImageSharp.
