# AR-0061: Раздача фотографий рецептов через публичный endpoint

Источник: ADR-0034

## Правило

Фотографии рецептов раздаются через HTTP-endpoint cookbook-сервиса. Endpoint публичный — авторизация для чтения не требуется. Ответ должен содержать заголовок `Cache-Control: public, max-age=86400`.

## Запрещено

- Требовать JWT для получения фотографии.
- Раздавать фотографии напрямую из файловой системы или через shared Docker volume.
