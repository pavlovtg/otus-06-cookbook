# AR-0004: Backend-сервисы не публикуются наружу напрямую

- **Домен**: rest-api
- **Статус**: активно

## Правило

В среде развёртывания (Docker Compose, любые целевые среды) backend-сервисы не имеют публичных портов наружу. Наружу публикуется только порт API Gateway. Backend-сервисы достижимы исключительно во внутренней сети между gateway и сервисами.

## Обоснование

AR-0003 запрещает клиентам обращаться к backend в обход gateway. Это правило является инфраструктурной гарантией AR-0003: даже при ошибке в клиенте или конфигурации фронтенда обращение к backend физически невозможно, потому что наружу опубликован только gateway.

## Нарушение

- В `docker-compose.yml` у backend-сервиса определён `ports:` с пробросом на хост.
- Backend-сервис размещён в общей с фронтендом/клиентами сети без посредничества gateway.
- Любой инструмент с хоста (curl, браузер) успешно открывает HTTP-эндпоинт backend без обращения к gateway.
