# AR-0003: Frontend взаимодействует с backend только через API Gateway

- **Домен**: rest-api
- **Статус**: активно

## Правило

Frontend и любые внешние потребители обращаются к backend-сервисам исключительно через API Gateway. Прямые сетевые вызовы из клиентского кода или внешних систем к backend-сервисам в обход gateway запрещены.

## Обоснование

API Gateway — единая точка доступа (ADR-0008): на нём централизованы маршрутизация, аутентификация, CORS, rate limiting и наблюдаемость. Прямые вызовы клиента к backend размывают границу системы, дублируют политики и привязывают клиент к внутренней топологии сервисов.

## Нарушение

- Frontend-код содержит URL backend-сервиса (имя контейнера, прямой порт) в обход gateway.
- Внешний потребитель документации или интеграции получает адрес backend-сервиса напрямую.
- Любой запрос «снаружи» успешно достигает backend, минуя gateway.
