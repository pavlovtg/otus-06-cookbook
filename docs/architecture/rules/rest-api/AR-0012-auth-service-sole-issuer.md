# AR-0012: Auth-service — единственный issuer JWT

Источник: ADR-0021, ADR-0022

## Правило

Все JWT-токены (пользовательские и сервисные) выпускаются исключительно сервисом `auth-service`. Доменные backend-сервисы и BFF не имеют права самостоятельно выпускать, перевыпускать или модифицировать JWT.

## Запрещено

- Выдача JWT из доменных сервисов или BFF.
- Подпись JWT внешними системами или вручную сформированными скриптами в обход `auth-service`.
- Хранение приватного ключа подписи где-либо, кроме конфигурации `auth-service` и сервисов, которым он нужен для валидации.
