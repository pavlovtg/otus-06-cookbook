# Workflow: create-prompt

## Алгоритм

1. Спросить у пользователя кодовое название промта (codename) — краткое, латиницей, через дефис.
2. Определить имя файла: `<index>-<codename>.md`, где `<index>` = максимальный существующий индекс в `docs/ai/prompts/` + 1, с ведущим нулём.
3. Создать файл `docs/ai/prompts/<index>-<codename>.md` с шаблоном ниже.

## Правила

- Язык: русский.
- Принципы экономии токенов (из `.clinerules/language.md`).

## Шаблон промта

```markdown
# <Название промта>

## Роль

- <Кто выполняет задачу>.

## Контекст

- <Что известно о ситуации>.

## Задача

- <Что нужно сделать>.

## Требования

- <Ограничения и критерии результата>.
```
