# Memory Bank

## Расположение
`docs/ai/memory-bank/`

## Файлы
- `projectbrief.md` — цель и стек проекта
- `productContext.md` — контекст продукта
- `activeContext.md` — текущая задача и прогресс
- `systemPatterns.md` — паттерны и соглашения
- `techContext.md` — технический стек и конфигурация

- **В начале каждой задачи** — прочитать все файлы memory bank.
- **В конце каждой задачи** — обновить `activeContext.md` и `progress.md`.
- Язык: русский.
- Использовать принципы экономии токенов (из `.clinerules/language.md`).
