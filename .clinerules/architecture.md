# Архитектура

## Правила

- В начале каждой задачи — прочитать `ARCHITECTURE.md` и следовать политике загрузки контекста.
- При создании/редактировании Markdown — соблюдать `docs/standards/markdown-code-style.md`.
- При любой операции с ADR — использовать skill `adr`.
- При любой операции с AR — использовать skill `ar`.
- При любой операции со стандартом — использовать skill `standard`.
- При любой операции с диаграммой — использовать skill `diagram`.
