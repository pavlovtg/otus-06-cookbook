# menu-planner

## Зачем

Пользователь не может планировать питание на неделю: нет инструмента, объединяющего рецепты из книги в недельную сетку.

## Что меняется

- Новая страница `/planner` — недельная сетка (пн–вс × завтрак/обед/ужин), drag-and-drop рецептов в слоты.
- Backend: новые эндпоинты для хранения плана меню пользователя.
- Навигация: пункт «Планировщик» добавляется в шапку (только для авторизованных).

## Возможности

### Новые возможности

- `meal-plan-get`: получить текущий план меню пользователя (7 дней × 3 приёма пищи)
- `meal-plan-update`: сохранить изменения плана (добавить/удалить блюдо, изменить порции)
- `meal-plan-clear`: очистить весь план меню

## Влияние

- **Backend**: новые эндпоинты в recipes-сервисе; новые таблицы `meal_plan_slots` и `meal_plan_items` в схеме `cookbook`.
- **Frontend**: новая страница `/planner`; новый BFF-файл `lib/bff/meal-plan.ts`; зависимость от `@dnd-kit/core` (уже зафиксирована в ADR-0036).
- **Навигация**: добавление пункта «Планировщик» в шапку с ролевой защитой (только авторизованные).
