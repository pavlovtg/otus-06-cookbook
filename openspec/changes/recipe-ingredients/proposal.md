# recipe-ingredients

## Зачем

Рецепты в системе не содержат ингредиентов — нельзя указать, из чего состоит блюдо и в каком количестве. Без этого невозможно реализовать масштабирование порций, список покупок и поиск по ингредиентам.

## Что меняется

- Рецепт получает список ингредиентов с количеством.
- Детальная страница рецепта отображает ингредиенты.
- Форма создания/редактирования рецепта позволяет добавлять и удалять ингредиенты.
- Удаление ингредиента из каталога блокируется, если он используется хотя бы в одном рецепте.

## Возможности

### Новые возможности

- `recipe-ingredient-add`: добавление ингредиентов к рецепту с количеством; отображение в карточке и детальной странице

### Изменяемые возможности

- `ingredient-delete`: добавляется сценарий блокировки удаления, если ингредиент используется в рецептах

## Влияние

- Модель данных рецепта (новая связь с ингредиентами).
- REST API рецептов (расширение запроса и ответа).
- UI: детальная страница и форма рецепта.
- Логика удаления ингредиента из каталога.
