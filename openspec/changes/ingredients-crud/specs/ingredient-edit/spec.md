## ADDED Requirements

### Requirement: Редактирование ингредиента

Система SHALL позволять редактировать существующий ингредиент через форму с предзаполненными значениями.

#### Scenario: Открытие формы редактирования

- **WHEN** пользователь нажимает кнопку редактирования у ингредиента
- **THEN** открывается форма с текущими значениями полей

#### Scenario: Успешное сохранение

- **WHEN** пользователь изменяет поля и сохраняет
- **THEN** ингредиент обновляется, список отражает новые значения

#### Scenario: Валидация при редактировании

- **WHEN** пользователь сохраняет форму с некорректными данными
- **THEN** отображаются ошибки валидации, данные не сохраняются

#### Scenario: Отмена редактирования

- **WHEN** пользователь нажимает «Отмена»
- **THEN** форма закрывается без изменений
