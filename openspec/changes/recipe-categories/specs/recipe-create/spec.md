## MODIFIED Requirements

### Requirement: Создание рецепта через форму

Система SHALL предоставлять форму создания рецепта с полями: название (обязательное), описание, время приготовления (мин, обязательное), сложность (обязательная), количество порций (обязательное), пошаговые инструкции, категории. Форма MUST валидировать данные на клиенте перед отправкой и отображать ошибки рядом с полями.

#### Scenario: Успешное создание рецепта

- **WHEN** пользователь заполняет форму корректными данными и нажимает «Сохранить»
- **THEN** рецепт сохраняется в системе и пользователь перенаправляется на детальную страницу нового рецепта

#### Scenario: Ошибка валидации на клиенте

- **WHEN** пользователь отправляет форму с незаполненным обязательным полем
- **THEN** форма не отправляется, рядом с полем отображается сообщение об ошибке

#### Scenario: Ошибка валидации на сервере

- **WHEN** сервер получает запрос с некорректными данными
- **THEN** API возвращает ошибку с описанием проблемы

#### Scenario: Создание рецепта с категориями

- **WHEN** пользователь выбирает категории и сохраняет рецепт
- **THEN** рецепт сохраняется с выбранными категориями
