# recipe-scale

## Зачем

На детальной странице рецепта количество ингредиентов отображается только для базового числа порций — пользователь не может пересчитать их под своё количество гостей. Функция масштабирования порций предусмотрена PRD-0001 и макетом, но не реализована.

## Что меняется

- Добавляется контрол изменения числа порций на детальной странице рецепта.
- Количество каждого ингредиента пересчитывается пропорционально при изменении порций.
- Пересчёт происходит на стороне клиента без обращения к серверу.

## Возможности

### Новые возможности

- `recipe-scale`: интерактивное масштабирование порций на детальной странице рецепта

### Изменяемые возможности

- `recipe-detail`: добавляется интерактивный контрол порций и пересчёт ингредиентов

## Влияние

- Детальная страница рецепта (frontend).
- Backend и API — без изменений.
