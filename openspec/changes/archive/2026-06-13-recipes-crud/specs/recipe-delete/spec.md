## ADDED Requirements

### Requirement: Удаление рецепта с подтверждением

Система SHALL запрашивать подтверждение перед удалением рецепта. После подтверждения рецепт MUST быть удалён из системы, пользователь перенаправляется на список рецептов.

#### Scenario: Успешное удаление после подтверждения

- **WHEN** пользователь нажимает «Удалить» и подтверждает действие в диалоге
- **THEN** рецепт удаляется из системы и пользователь перенаправляется на список рецептов

#### Scenario: Отмена удаления

- **WHEN** пользователь нажимает «Удалить», но отменяет действие в диалоге
- **THEN** рецепт остаётся в системе, пользователь остаётся на текущей странице

#### Scenario: Удаление несуществующего рецепта

- **WHEN** API получает запрос на удаление несуществующего рецепта
- **THEN** API возвращает `404 Not Found` с `application/problem+json`
