## MODIFIED Requirements

### Requirement: Детальная страница рецепта

Детальная страница SHALL отображать полную информацию о рецепте, включая категории в виде тегов рядом с другими метаданными.

#### Scenario: Отображение рецепта с категориями

- **WHEN** пользователь открывает детальную страницу рецепта с категориями
- **THEN** страница отображает категории как теги

#### Scenario: Отображение рецепта без категорий

- **WHEN** пользователь открывает детальную страницу рецепта без категорий
- **THEN** страница отображается без тегов категорий
