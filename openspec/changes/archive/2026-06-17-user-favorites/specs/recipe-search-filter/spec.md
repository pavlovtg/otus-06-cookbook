## ADDED Requirements

### Requirement: Поиск работает внутри подмножества избранных рецептов

При активном режиме «Избранное» поиск SHALL применяться только к рецептам, добавленным пользователем в избранное. Логика поиска (AND по словам, поля поиска) MUST оставаться неизменной.

#### Scenario: Поиск в режиме «Избранное» с совпадениями

- **WHEN** авторизованный пользователь вводит поисковый запрос в режиме «Избранное»
- **THEN** система возвращает только те избранные рецепты, которые соответствуют запросу

#### Scenario: Поиск в режиме «Избранное» без совпадений

- **WHEN** авторизованный пользователь вводит поисковый запрос в режиме «Избранное» и ни один избранный рецепт не соответствует запросу
- **THEN** система возвращает пустой список

#### Scenario: Поиск в режиме «Все рецепты» не ограничен избранным

- **WHEN** авторизованный пользователь вводит поисковый запрос в режиме «Все рецепты»
- **THEN** система ищет по всем доступным рецептам, не ограничиваясь избранным
