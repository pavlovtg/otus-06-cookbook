## ADDED Requirements

### Requirement: Авторизованный пользователь добавляет рецепт в избранное

Авторизованный пользователь SHALL иметь возможность добавить рецепт в избранное. Повторное добавление уже избранного рецепта MUST не создавать дубликат.

#### Scenario: Добавление рецепта в избранное

- **WHEN** авторизованный пользователь добавляет рецепт в избранное
- **THEN** рецепт появляется в его списке избранного

#### Scenario: Повторное добавление уже избранного рецепта

- **WHEN** авторизованный пользователь добавляет в избранное рецепт, который уже там есть
- **THEN** операция завершается успешно без создания дубликата

#### Scenario: Неавторизованный пользователь не может добавить в избранное

- **WHEN** неавторизованный пользователь пытается добавить рецепт в избранное
- **THEN** система отказывает в доступе

### Requirement: Авторизованный пользователь удаляет рецепт из избранного

Авторизованный пользователь SHALL иметь возможность убрать рецепт из избранного. Удаление рецепта, которого нет в избранном, MUST завершаться успешно.

#### Scenario: Удаление рецепта из избранного

- **WHEN** авторизованный пользователь убирает рецепт из избранного
- **THEN** рецепт исчезает из его списка избранного

#### Scenario: Удаление рецепта, которого нет в избранном

- **WHEN** авторизованный пользователь пытается убрать из избранного рецепт, которого там нет
- **THEN** операция завершается успешно

#### Scenario: Неавторизованный пользователь не может удалить из избранного

- **WHEN** неавторизованный пользователь пытается убрать рецепт из избранного
- **THEN** система отказывает в доступе
