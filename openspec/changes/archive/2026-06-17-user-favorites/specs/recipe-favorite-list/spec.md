## ADDED Requirements

### Requirement: Авторизованный пользователь просматривает список избранных рецептов

Авторизованный пользователь SHALL иметь возможность получить список только тех рецептов, которые он добавил в избранное. Список SHALL поддерживать пагинацию и поиск по тем же правилам, что и общий список рецептов.

#### Scenario: Просмотр избранного

- **WHEN** авторизованный пользователь запрашивает список рецептов в режиме «Избранное»
- **THEN** система возвращает только рецепты, добавленные этим пользователем в избранное

#### Scenario: Поиск внутри избранного

- **WHEN** авторизованный пользователь вводит поисковый запрос в режиме «Избранное»
- **THEN** система возвращает только те избранные рецепты, которые соответствуют запросу

#### Scenario: Пустое избранное

- **WHEN** авторизованный пользователь запрашивает список избранного, не добавив ни одного рецепта
- **THEN** система возвращает пустой список

#### Scenario: Неавторизованный пользователь не может просматривать избранное

- **WHEN** неавторизованный пользователь запрашивает список рецептов в режиме «Избранное»
- **THEN** система отказывает в доступе
