# user-favorites

## Зачем

Авторизованный пользователь не может сохранять понравившиеся рецепты — каждый раз приходится искать их заново. Функция «Избранное» позволяет быстро возвращаться к нужным рецептам и фильтровать список только по ним.

## Что меняется

- Авторизованный пользователь может добавить или убрать рецепт из избранного.
- На карточке рецепта и на детальной странице отображается иконка-сердечко; для неавторизованных — скрыта.
- В сайдбаре страницы рецептов появляется переключатель «Все рецепты» / «Избранное».
- При активном режиме «Избранное» список показывает только сохранённые рецепты; поиск работает внутри этого подмножества.
- Избранное хранится на сервере, привязано к аккаунту пользователя.
- Демо-данные включают несколько записей избранного для пользователей `ivlev@cookbook.local` и `renat@cookbook.local`.

## Возможности

### Новые возможности

- `recipe-favorite-toggle`: добавление/удаление рецепта из избранного
- `recipe-favorite-list`: просмотр списка рецептов в режиме «Избранное»

### Изменяемые возможности

- `recipe-list`: карточка рецепта показывает признак «в избранном»; добавляется переключатель режима в сайдбаре
- `recipe-search-filter`: поиск работает внутри подмножества избранных при активном режиме «Избранное»

## Влияние

- Страница списка рецептов: новый переключатель в сайдбаре, иконка на карточке и детальной странице.
- API рецептов: новые эндпоинты для управления избранным; расширение списка рецептов параметром фильтрации.
- Хранилище: новая таблица для хранения избранного пользователей.
