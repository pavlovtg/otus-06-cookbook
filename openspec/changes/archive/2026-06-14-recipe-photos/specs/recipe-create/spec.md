## ADDED Requirements

### Requirement: Загрузка фото при создании рецепта

Форма создания рецепта SHALL предоставлять возможность загрузить фото. Загрузка фото необязательна — рецепт можно создать без фото.

#### Scenario: Создание рецепта с фото

- **WHEN** пользователь заполняет форму, прикрепляет фото и нажимает «Сохранить»
- **THEN** рецепт создаётся с фото, которое отображается на детальной странице

#### Scenario: Создание рецепта без фото

- **WHEN** пользователь заполняет форму без прикрепления фото и нажимает «Сохранить»
- **THEN** рецепт создаётся без фото, на детальной странице отображается заглушка
