## ADDED Requirements

### Requirement: Управление фото при редактировании рецепта

При редактировании рецепта пользователь SHALL иметь возможность заменить или удалить фото независимо от изменения остальных полей. Сохранение изменений в полях рецепта не затрагивает фото.

#### Scenario: Замена фото при редактировании

- **WHEN** пользователь нажимает «Заменить фото» и выбирает новый файл
- **THEN** старое фото заменяется новым без необходимости сохранять остальные поля формы

#### Scenario: Удаление фото при редактировании

- **WHEN** пользователь нажимает «Удалить фото» и подтверждает действие
- **THEN** фото удаляется, на месте фото отображается заглушка

#### Scenario: Сохранение формы без изменения фото

- **WHEN** пользователь изменяет поля рецепта и нажимает «Сохранить», не трогая фото
- **THEN** изменения в полях сохраняются, фото остаётся прежним
