## ADDED Requirements

### Requirement: Запуск приложения одной командой

Система SHALL запускаться в чистом окружении командой `docker compose up` без дополнительной настройки. После запуска приложение MUST быть доступно в браузере.

#### Scenario: Первый запуск в чистом окружении

- **WHEN** пользователь выполняет `docker compose up` в корне репозитория
- **THEN** все сервисы стартуют и приложение доступно в браузере

#### Scenario: Повторный запуск

- **WHEN** пользователь повторно выполняет `docker compose up`
- **THEN** приложение запускается без ошибок, данные сохранены
