## ADDED Requirements

### Requirement: Защита маршрутов создания и редактирования рецептов

Система SHALL ограничивать доступ к страницам создания и редактирования рецептов только для авторизованных пользователей.

#### Scenario: Неавторизованный пользователь открывает страницу создания рецепта

- **WHEN** неавторизованный пользователь переходит на страницу создания рецепта
- **THEN** система перенаправляет его на страницу входа

#### Scenario: Неавторизованный пользователь открывает страницу редактирования рецепта

- **WHEN** неавторизованный пользователь переходит на страницу редактирования рецепта
- **THEN** система перенаправляет его на страницу входа

#### Scenario: Авторизованный пользователь открывает страницу создания рецепта

- **WHEN** авторизованный пользователь переходит на страницу создания рецепта
- **THEN** страница отображается без перенаправления
