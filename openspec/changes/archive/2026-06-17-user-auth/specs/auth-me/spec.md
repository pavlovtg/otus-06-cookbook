## ADDED Requirements

### Requirement: Профиль текущего пользователя

Система SHALL возвращать данные текущего авторизованного пользователя: идентификатор, email, отображаемое имя и роль.

#### Scenario: Получение профиля авторизованным пользователем

- **WHEN** авторизованный пользователь запрашивает свой профиль
- **THEN** система возвращает его данные (id, email, displayName, role)

#### Scenario: Запрос без авторизации

- **WHEN** неавторизованный пользователь запрашивает профиль
- **THEN** система возвращает ошибку 401
