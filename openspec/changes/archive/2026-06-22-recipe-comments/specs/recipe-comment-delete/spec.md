## ADDED Requirements

### Requirement: Удаление комментария

Автор комментария SHALL иметь возможность удалить свой комментарий. Автор рецепта SHALL иметь возможность удалить любой комментарий к своему рецепту. Администратор SHALL иметь возможность удалить любой комментарий. Остальные авторизованные пользователи SHALL не иметь возможности удалить чужой комментарий. Неавторизованный пользователь SHALL не иметь возможности удалить комментарий.

#### Scenario: Автор комментария удаляет свой комментарий

- **WHEN** авторизованный пользователь, являющийся автором комментария, запрашивает его удаление
- **THEN** комментарий удаляется

#### Scenario: Автор рецепта удаляет чужой комментарий

- **WHEN** авторизованный пользователь, являющийся автором рецепта, запрашивает удаление комментария другого пользователя
- **THEN** комментарий удаляется

#### Scenario: Администратор удаляет любой комментарий

- **WHEN** администратор запрашивает удаление любого комментария
- **THEN** комментарий удаляется

#### Scenario: Обычный пользователь не может удалить чужой комментарий

- **WHEN** авторизованный пользователь, не являющийся ни автором комментария, ни автором рецепта, ни администратором, запрашивает удаление комментария
- **THEN** система отклоняет запрос с ошибкой доступа

#### Scenario: Удаление несуществующего комментария

- **WHEN** авторизованный пользователь запрашивает удаление несуществующего комментария
- **THEN** система отклоняет запрос с ошибкой

#### Scenario: Неавторизованный пользователь не может удалить комментарий

- **WHEN** неавторизованный пользователь пытается удалить комментарий
- **THEN** система отклоняет запрос с ошибкой авторизации
