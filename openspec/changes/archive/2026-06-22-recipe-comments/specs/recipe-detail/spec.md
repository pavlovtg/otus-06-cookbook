## MODIFIED Requirements

### Requirement: Детальная страница рецепта

Детальная страница SHALL отображать полную информацию о рецепте, включая категории в виде тегов, имя автора и признак публичности. Блок ингредиентов SHALL содержать интерактивный контрол масштабирования порций. Приватный рецепт SHALL быть доступен только автору и администратору. Детальная страница SHALL отображать секцию комментариев со списком всех комментариев к рецепту. Авторизованный пользователь, не оставивший комментарий, SHALL видеть форму добавления комментария. Авторизованный пользователь, уже оставивший комментарий, SHALL видеть сообщение об этом вместо формы. Неавторизованный пользователь SHALL не видеть форму добавления комментария.

#### Scenario: Открытие детальной страницы

- **WHEN** пользователь переходит по ссылке на рецепт
- **THEN** отображается детальная страница с полными данными рецепта, включая имя автора и признак публичности

#### Scenario: Рецепт не найден

- **WHEN** пользователь открывает URL несуществующего рецепта
- **THEN** отображается страница с сообщением «Рецепт не найден»

#### Scenario: Чужой приватный рецепт недоступен

- **WHEN** пользователь открывает URL чужого приватного рецепта
- **THEN** отображается страница с сообщением об ошибке доступа

#### Scenario: Секция комментариев отображается на детальной странице

- **WHEN** пользователь открывает детальную страницу рецепта
- **THEN** отображается секция комментариев со списком всех комментариев (имя автора, дата, текст)

#### Scenario: Форма добавления комментария для авторизованного пользователя без комментария

- **WHEN** авторизованный пользователь, не оставивший комментарий к рецепту, открывает детальную страницу
- **THEN** отображается форма добавления комментария

#### Scenario: Форма скрыта если пользователь уже оставил комментарий

- **WHEN** авторизованный пользователь, уже оставивший комментарий к рецепту, открывает детальную страницу
- **THEN** форма добавления комментария не отображается

#### Scenario: Неавторизованный пользователь не видит форму комментария

- **WHEN** неавторизованный пользователь открывает детальную страницу рецепта
- **THEN** форма добавления комментария не отображается
