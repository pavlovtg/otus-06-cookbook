## ADDED Requirements

### Requirement: Получение списка комментариев к рецепту

Система SHALL предоставлять возможность получить комментарии к рецепту без авторизации. Список SHALL поддерживать пагинацию с размером страницы по умолчанию 10 комментариев. Каждый комментарий SHALL содержать имя автора, текст и дату создания. Комментарии SHALL быть отсортированы по дате создания в порядке убывания (новые первыми).

#### Scenario: Получение первой страницы комментариев

- **WHEN** пользователь запрашивает список комментариев к рецепту без указания страницы
- **THEN** система возвращает первые 10 комментариев, отсортированных по дате создания (новые первыми), и общее количество комментариев

#### Scenario: Получение второй страницы комментариев

- **WHEN** пользователь запрашивает вторую страницу комментариев к рецепту с более чем 10 комментариями
- **THEN** система возвращает следующие 10 комментариев

#### Scenario: Рецепт без комментариев

- **WHEN** пользователь запрашивает список комментариев к рецепту, у которого нет комментариев
- **THEN** система возвращает пустой список и общее количество 0

#### Scenario: Комментарии к несуществующему рецепту

- **WHEN** пользователь запрашивает список комментариев к несуществующему рецепту
- **THEN** система отклоняет запрос с ошибкой

#### Scenario: Каждый комментарий содержит имя автора

- **WHEN** пользователь получает список комментариев
- **THEN** каждый комментарий содержит отображаемое имя автора
