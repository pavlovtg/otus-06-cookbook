## ADDED Requirements

### Requirement: Удаление категории

Система SHALL удалять категорию только если она не присвоена ни одному рецепту.

#### Scenario: Успешное удаление

- **WHEN** пользователь удаляет категорию, которая не используется в рецептах
- **THEN** категория удаляется из системы

#### Scenario: Категория используется в рецептах

- **WHEN** пользователь пытается удалить категорию, присвоенную хотя бы одному рецепту
- **THEN** система отклоняет запрос с ошибкой

#### Scenario: Категория не найдена

- **WHEN** пользователь пытается удалить несуществующую категорию
- **THEN** система отклоняет запрос с ошибкой
