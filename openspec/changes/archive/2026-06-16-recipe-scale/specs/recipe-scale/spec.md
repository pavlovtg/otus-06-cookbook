## ADDED Requirements

### Requirement: Масштабирование порций

Детальная страница рецепта SHALL предоставлять контрол изменения числа порций. При изменении числа порций количество каждого ингредиента SHALL пересчитываться пропорционально: `amount × (targetServings / baseServings)`. Минимальное значение порций — 1, максимальное — 99.

#### Scenario: Увеличение числа порций

- **WHEN** пользователь нажимает «+» в контроле порций
- **THEN** число порций увеличивается на 1 и количества ингредиентов пересчитываются пропорционально

#### Scenario: Уменьшение числа порций

- **WHEN** пользователь нажимает «−» в контроле порций
- **THEN** число порций уменьшается на 1 и количества ингредиентов пересчитываются пропорционально

#### Scenario: Минимальное значение порций

- **WHEN** текущее число порций равно 1 и пользователь нажимает «−»
- **THEN** число порций остаётся равным 1

#### Scenario: Максимальное значение порций

- **WHEN** текущее число порций равно 99 и пользователь нажимает «+»
- **THEN** число порций остаётся равным 99

#### Scenario: Начальное значение порций

- **WHEN** пользователь открывает детальную страницу рецепта
- **THEN** контрол порций показывает базовое значение `servings` из рецепта
