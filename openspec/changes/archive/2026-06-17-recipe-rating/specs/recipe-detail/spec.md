## ADDED Requirements

### Requirement: Детальная страница рецепта показывает рейтинг

Детальная страница рецепта SHALL отображать поле `averageRating` (средний рейтинг).
Для авторизованного пользователя ответ SHALL также содержать поле `myRating` (личная оценка, если выставлена).
Поле `averageRating` SHALL быть `null` если рецепт не имеет оценок.
Поле `myRating` SHALL быть `null` если пользователь не оценивал рецепт или не авторизован.

#### Scenario: Детальная страница содержит поля рейтинга

- **WHEN** клиент запрашивает детальную страницу рецепта
- **THEN** ответ содержит поля `averageRating` и `myRating`

#### Scenario: Рецепт без оценок на детальной странице

- **WHEN** клиент запрашивает детальную страницу рецепта без оценок
- **THEN** поле `averageRating` равно `null`

#### Scenario: Поле myRating для авторизованного пользователя с оценкой

- **WHEN** авторизованный пользователь, оценивший рецепт, запрашивает детальную страницу
- **THEN** поле `myRating` содержит его оценку (1–5)
