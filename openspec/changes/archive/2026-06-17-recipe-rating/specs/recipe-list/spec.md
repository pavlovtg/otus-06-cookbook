## ADDED Requirements

### Requirement: Карточка рецепта показывает рейтинг

Карточка рецепта в списке SHALL отображать поле `averageRating` (средний рейтинг).
Для авторизованного пользователя карточка SHALL также отображать поле `myRating` (личная оценка, если выставлена).
Поле `averageRating` SHALL быть `null` если рецепт не имеет оценок.
Поле `myRating` SHALL быть `null` если пользователь не оценивал рецепт или не авторизован.

#### Scenario: Список рецептов содержит поля рейтинга

- **WHEN** клиент запрашивает список рецептов
- **THEN** каждый элемент ответа содержит поля `averageRating` и `myRating`

#### Scenario: Рецепт без оценок в списке

- **WHEN** клиент запрашивает список рецептов и рецепт не имеет оценок
- **THEN** поле `averageRating` для этого рецепта равно `null`

#### Scenario: Поле myRating для неавторизованного пользователя

- **WHEN** неавторизованный клиент запрашивает список рецептов
- **THEN** поле `myRating` для всех рецептов равно `null`
