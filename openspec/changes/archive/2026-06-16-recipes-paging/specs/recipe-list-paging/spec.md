## ADDED Requirements

### Requirement: Пагинация списка рецептов

Список рецептов SHALL отображаться постранично. На одной странице MUST быть не более 18 рецептов. Система MUST знать общее количество рецептов.

#### Scenario: Список разбит на страницы

- **WHEN** рецептов больше 18
- **THEN** отображается только 18 рецептов на текущей странице

#### Scenario: Последняя страница с неполным набором

- **WHEN** на последней странице рецептов меньше 18
- **THEN** отображаются только оставшиеся рецепты

### Requirement: Навигация по страницам

UI SHALL отображать элементы навигации для перехода между страницами списка рецептов.

#### Scenario: Навигация отображается при нескольких страницах

- **WHEN** рецептов больше 18
- **THEN** под списком отображаются элементы навигации по страницам

#### Scenario: Переход на следующую страницу

- **WHEN** пользователь нажимает кнопку следующей страницы
- **THEN** отображается следующая страница рецептов

#### Scenario: Навигация не отображается при одной странице

- **WHEN** рецептов не более 18
- **THEN** элементы навигации не отображаются

### Requirement: Индикация общего количества рецептов

UI SHALL отображать общее количество рецептов в системе.

#### Scenario: Счётчик рецептов виден

- **WHEN** пользователь открывает список рецептов
- **THEN** отображается общее количество рецептов
