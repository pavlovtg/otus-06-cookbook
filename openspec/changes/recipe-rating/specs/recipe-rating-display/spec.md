## ADDED Requirements

### Requirement: Отображение рейтинга на карточке рецепта

Карточка рецепта в списке SHALL отображать средний рейтинг рецепта (среднее арифметическое всех оценок).
Средний рейтинг SHALL быть виден всем пользователям, включая неавторизованных.
Если рецепт не имеет оценок, SHALL отображаться прочерк или нулевое значение.
Для авторизованного пользователя, уже оценившего рецепт, SHALL дополнительно отображаться его личная оценка.

#### Scenario: Карточка рецепта с оценками для неавторизованного пользователя

- **WHEN** неавторизованный пользователь открывает список рецептов
- **THEN** на каждой карточке отображается средний рейтинг рецепта

#### Scenario: Карточка рецепта без оценок

- **WHEN** пользователь открывает список рецептов и рецепт не имеет оценок
- **THEN** на карточке отображается прочерк или нулевое значение вместо рейтинга

#### Scenario: Карточка рецепта для авторизованного пользователя с личной оценкой

- **WHEN** авторизованный пользователь, уже оценивший рецепт, открывает список рецептов
- **THEN** на карточке отображается средний рейтинг и личная оценка пользователя

### Requirement: Отображение рейтинга на детальной странице рецепта

Детальная страница рецепта SHALL отображать средний рейтинг рецепта.
Для авторизованного пользователя SHALL отображаться интерактивный виджет звёзд для выставления оценки.
Текущая оценка пользователя SHALL быть визуально выделена в виджете.
Для неавторизованного пользователя виджет выставления оценки SHALL быть скрыт.

#### Scenario: Детальная страница для неавторизованного пользователя

- **WHEN** неавторизованный пользователь открывает детальную страницу рецепта
- **THEN** отображается средний рейтинг, виджет выставления оценки не отображается

#### Scenario: Детальная страница для авторизованного пользователя без оценки

- **WHEN** авторизованный пользователь, не оценивавший рецепт, открывает детальную страницу
- **THEN** отображается средний рейтинг и интерактивный виджет звёзд без выделенной оценки

#### Scenario: Детальная страница для авторизованного пользователя с оценкой

- **WHEN** авторизованный пользователь, уже оценивший рецепт, открывает детальную страницу
- **THEN** отображается средний рейтинг и виджет звёзд с выделенной личной оценкой пользователя
