## Requirements

### Requirement: Автодополнение поискового запроса

Система SHALL предоставлять подсказки автодополнения при вводе поискового запроса.
Подсказки формируются из названий категорий и ингредиентов, совпадающих с последним словом запроса.
Автодополнение активируется при длине последнего слова ≥ 2 символов.

#### Scenario: Подсказки по категориям

- **WHEN** пользователь вводит строку, последнее слово которой совпадает с началом названия категории
- **THEN** система показывает список подходящих категорий с пометкой «категория»

#### Scenario: Подсказки по ингредиентам

- **WHEN** пользователь вводит строку, последнее слово которой совпадает с началом названия ингредиента
- **THEN** система показывает список подходящих ингредиентов с пометкой «ингредиент»

#### Scenario: Выбор подсказки

- **WHEN** пользователь выбирает подсказку из списка
- **THEN** выбранное слово подставляется в поисковую строку вместо последнего введённого слова

#### Scenario: Короткий ввод

- **WHEN** последнее слово запроса содержит менее 2 символов
- **THEN** список подсказок не отображается
