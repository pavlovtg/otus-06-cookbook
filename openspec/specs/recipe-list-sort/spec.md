## Requirements

### Requirement: Сортировка списка рецептов по названию

Система SHALL поддерживать сортировку списка рецептов по параметру `sort`.
Допустимые значения: `title_asc` (по алфавиту, А→Я), `title_desc` (по алфавиту, Я→А) и `rating_desc` (по убыванию среднего рейтинга).
При отсутствии параметра применяется сортировка по умолчанию (`title_asc`).

#### Scenario: Сортировка по алфавиту возрастание

- **WHEN** клиент передаёт параметр `sort=title_asc`
- **THEN** система возвращает рецепты, отсортированные по названию от А до Я

#### Scenario: Сортировка по алфавиту убывание

- **WHEN** клиент передаёт параметр `sort=title_desc`
- **THEN** система возвращает рецепты, отсортированные по названию от Я до А

#### Scenario: Сортировка по умолчанию

- **WHEN** клиент не передаёт параметр `sort`
- **THEN** система возвращает рецепты, отсортированные по названию от А до Я

#### Scenario: Сортировка совместно с поиском

- **WHEN** клиент передаёт параметры `q=курица` и `sort=title_desc`
- **THEN** система возвращает отфильтрованные рецепты, отсортированные по названию от Я до А

#### Scenario: Сортировка по рейтингу убывание

- **WHEN** клиент передаёт параметр `sort=rating_desc`
- **THEN** система возвращает рецепты, отсортированные по убыванию среднего рейтинга; рецепты без оценок идут последними
