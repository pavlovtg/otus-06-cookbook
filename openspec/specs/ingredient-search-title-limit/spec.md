### Requirement: Ограничение длины строки поиска по title

Система SHALL ограничивать параметр `title` в `GET /api/v1/ingredients` максимальной длиной 200 символов.
При превышении лимита система SHALL вернуть `400`.
UI SHALL ограничивать ввод в поле поиска до 200 символов.

#### Scenario: Поиск с допустимой длиной title

- **WHEN** клиент вызывает `GET /api/v1/ingredients?title=Картофель`
- **THEN** система возвращает `200` с отфильтрованным списком

#### Scenario: Поиск с title длиннее 200 символов

- **WHEN** клиент вызывает `GET /api/v1/ingredients?title=<строка длиннее 200 символов>`
- **THEN** система возвращает `400`

#### Scenario: UI ограничивает ввод в поле поиска

- **WHEN** пользователь вводит текст в поле поиска на странице `/ingredients`
- **THEN** поле принимает не более 200 символов
