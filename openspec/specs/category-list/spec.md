## ADDED Requirements

### Requirement: Просмотр списка категорий

Система SHALL предоставлять полный список всех категорий. Каждая категория содержит название, описание и тип.

#### Scenario: Список категорий доступен

- **WHEN** пользователь запрашивает список категорий
- **THEN** система возвращает все категории

#### Scenario: Нет категорий

- **WHEN** пользователь запрашивает список категорий и категорий нет
- **THEN** система возвращает пустой список

### Requirement: Группировка категорий по типу

Система SHALL отображать категории, сгруппированные по типу (7 групп: роль в приёме пищи, способ приготовления, основной ингредиент, национальная кухня, время употребления, диетические особенности, форма подачи).

#### Scenario: Категории сгруппированы

- **WHEN** пользователь открывает страницу категорий
- **THEN** категории отображаются в группах по типу
