## ADDED Requirements

### Requirement: Редактирование категории

Система SHALL позволять изменять название и описание категории. Тип категории MUST оставаться неизменным после создания — любое переданное значение типа игнорируется.

#### Scenario: Успешное редактирование

- **WHEN** пользователь обновляет название или описание существующей категории
- **THEN** изменения сохраняются

#### Scenario: Попытка изменить тип

- **WHEN** пользователь передаёт новое значение типа при редактировании
- **THEN** тип остаётся прежним, остальные поля обновляются

#### Scenario: Категория не найдена

- **WHEN** пользователь пытается редактировать несуществующую категорию
- **THEN** система отклоняет запрос с ошибкой
