### Requirement: Рецепт содержит список ингредиентов

Рецепт SHALL содержать список ингредиентов, каждый из которых включает ссылку на ингредиент из каталога и количество.

#### Scenario: Создание рецепта с ингредиентами

- **WHEN** пользователь создаёт рецепт и указывает список ингредиентов с количеством
- **THEN** рецепт сохраняется с привязанными ингредиентами

#### Scenario: Создание рецепта без ингредиентов

- **WHEN** пользователь создаёт рецепт без указания ингредиентов
- **THEN** рецепт сохраняется с пустым списком ингредиентов

#### Scenario: Обновление ингредиентов рецепта

- **WHEN** пользователь редактирует рецепт и изменяет список ингредиентов
- **THEN** список ингредиентов рецепта обновляется согласно новым данным

### Requirement: Количество ингредиента в рецепте валидируется

Количество ингредиента SHALL быть положительным числом больше 0, не более 100 000, с точностью до 3 знаков после запятой.

#### Scenario: Некорректное количество — ноль или отрицательное

- **WHEN** пользователь указывает количество ингредиента равное 0 или отрицательное
- **THEN** система возвращает ошибку валидации

#### Scenario: Некорректное количество — превышение максимума

- **WHEN** пользователь указывает количество ингредиента больше 100 000
- **THEN** система возвращает ошибку валидации

### Requirement: Количество ингредиентов в рецепте ограничено

Рецепт SHALL содержать не более 50 ингредиентов.

#### Scenario: Превышение лимита ингредиентов

- **WHEN** пользователь пытается сохранить рецепт с более чем 50 ингредиентами
- **THEN** система возвращает ошибку валидации

### Requirement: Детальная страница рецепта отображает ингредиенты

Детальная страница рецепта SHALL отображать список ингредиентов с названием, количеством и единицей измерения.

#### Scenario: Рецепт с ингредиентами

- **WHEN** пользователь открывает детальную страницу рецепта, у которого есть ингредиенты
- **THEN** отображается список ингредиентов с названием, количеством и единицей измерения

#### Scenario: Рецепт без ингредиентов

- **WHEN** пользователь открывает детальную страницу рецепта без ингредиентов
- **THEN** секция ингредиентов не отображается или показывает пустое состояние

### Requirement: Форма рецепта позволяет добавлять и убирать ингредиенты

Форма создания и редактирования рецепта SHALL позволять добавлять ингредиенты из каталога с указанием количества, а также убирать ингредиент из списка рецепта.

#### Scenario: Добавление ингредиента в форме

- **WHEN** пользователь выбирает ингредиент из каталога и указывает количество
- **THEN** ингредиент добавляется в список ингредиентов рецепта в форме

#### Scenario: Убирание ингредиента из рецепта в форме

- **WHEN** пользователь нажимает кнопку «убрать» рядом с ингредиентом в форме рецепта
- **THEN** ингредиент убирается из списка ингредиентов данного рецепта в форме
