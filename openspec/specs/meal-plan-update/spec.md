## Requirements

### Requirement: Добавление блюда в слот

Авторизованный пользователь SHALL добавить рецепт в любой слот недельной сетки перетаскиванием.
В один слот можно добавить несколько блюд, в том числе одно и то же блюдо несколько раз.
После добавления план SHALL сохраняться автоматически без явного действия пользователя.

#### Scenario: Пользователь перетаскивает рецепт в слот

- **WHEN** авторизованный пользователь перетаскивает карточку рецепта из панели в слот сетки
- **THEN** рецепт появляется в слоте с количеством порций по умолчанию, план сохраняется

#### Scenario: Несколько блюд в одном слоте

- **WHEN** авторизованный пользователь перетаскивает второй рецепт в уже заполненный слот
- **THEN** оба блюда отображаются в слоте

### Requirement: Изменение порций блюда в слоте

Авторизованный пользователь SHALL изменить количество порций для любого блюда в слоте.
Количество порций SHALL быть целым числом от 1 до 99.
После изменения план SHALL сохраняться автоматически.

#### Scenario: Пользователь меняет количество порций

- **WHEN** авторизованный пользователь изменяет значение порций для блюда в слоте
- **THEN** новое значение сохраняется, план обновляется

### Requirement: Удаление блюда из слота

Авторизованный пользователь SHALL удалить отдельное блюдо из слота.
После удаления план SHALL сохраняться автоматически.

#### Scenario: Пользователь удаляет блюдо из слота

- **WHEN** авторизованный пользователь нажимает кнопку удаления рядом с блюдом в слоте
- **THEN** блюдо исчезает из слота, план сохраняется
