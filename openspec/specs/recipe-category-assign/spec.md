## Requirements

### Requirement: Назначение категорий рецепту

При создании и редактировании рецепта пользователь SHALL иметь возможность выбрать категории из справочника. Рецепту можно назначить любое количество категорий, но не более одной категории одного типа.

#### Scenario: Выбор категории при создании рецепта

- **WHEN** пользователь вводит текст в поле выбора категорий
- **THEN** система показывает подходящие категории из справочника

#### Scenario: Добавление категории

- **WHEN** пользователь выбирает категорию из списка подсказок
- **THEN** категория добавляется как тег под полем ввода

#### Scenario: Удаление категории

- **WHEN** пользователь нажимает крестик на теге категории
- **THEN** категория удаляется из выбранных

#### Scenario: Замена категории того же типа

- **WHEN** пользователь выбирает категорию типа, уже присвоенного рецепту
- **THEN** новая категория заменяет существующую категорию того же типа

#### Scenario: Сохранение рецепта с категориями

- **WHEN** пользователь сохраняет рецепт с выбранными категориями
- **THEN** рецепт сохраняется с указанными категориями

#### Scenario: Сохранение рецепта без категорий

- **WHEN** пользователь сохраняет рецепт, не выбрав ни одной категории
- **THEN** рецепт сохраняется без категорий
